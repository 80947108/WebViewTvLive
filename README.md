## FAQ
**Q: 在电视机/盒子上无法自动全屏？**
<br/>
A: 一般是设备 WebView 的问题，待X5内核下载完成，重启 App 即可。若无法自动下载X5内核，请在设置-TBS调试界面内手动安装。
<br/>
**Q: 我打开怎么只有CCTV-1？**
<br/>
A: 频道列表托管在Github，出现此问题请检查您与Github之间的连接。
